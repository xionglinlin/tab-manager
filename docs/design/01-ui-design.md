# UI 设计文档

## 设计理念

### 核心原则
1. **简洁美观** - 现代化 UI 设计，视觉舒适
2. **高效实用** - 快速找到目标标签页，减少操作步骤
3. **智能分组** - 自动分类，减少用户思考负担
4. **轻量级** - 不影响浏览器性能

### 设计风格
- **深色主题为主**，护眼且现代
- **卡片式域名分组**，视觉清晰
- **列表式标签页**，信息密度高
- **悬浮操作按钮**，界面干净
- **好看的点击反馈**，但不花哨

### DomainCard.vue - 域名卡片

**布局**：
```

│  🟢 GitHub 5 ⭐                        [×] [📌] [🔄]      │
├─────────────────────────────────────────────────────────────┤
```

**样式**：

- 边框：1px solid var(--border-primary)

- 点击复制按钮：复制该标签页链接
- 点击关闭按钮：关闭该标签页
- 点击卡片操作按钮：执行对应操作（关闭全部、置顶、去重）

## 动画设计

### 入场动画

```css
/* 淡入上移 */
@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(12px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

/* 应用 */
.header { animation: fadeInUp 0.2s ease both; }
.stats-bar { animation: fadeInUp 0.2s ease 100ms both; }
.search-bar { animation: fadeInUp 0.2s ease 200ms both; }
.dashboard { animation: fadeInUp 0.2s ease 300ms both; }
```

### 交互动画

```css
/* 关闭标签页 */
.tab-item {
  transition: opacity 0.2s, transform 0.2s;
}

.tab-item.closing {
  opacity: 0;
  transform: translateX(-20px);
}

/* 关闭域名卡片 */
.domain-card {
  transition: opacity 0.3s, transform 0.3s;
}

.domain-card.closing {
  opacity: 0;
  transform: scale(0.96);
}

/* 悬浮效果 */
.domain-card:hover {
  transform: translateY(-2px);
  box-shadow: var(--shadow-md);
}
```
