# 测试指南

## 测试环境准备

### 1. 构建项目

```bash
cd tab-manager
npm run build
```

### 2. 加载扩展到 Chrome

1. 打开 Chrome 浏览器
2. 在地址栏输入 `chrome://extensions` 并回车
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `tab-manager/dist` 目录
6. 扩展加载成功后，会在工具栏显示图标

### 3. 固定扩展图标

1. 点击工具栏上的扩展图标（拼图形状）
2. 找到"Tab Manager"
3. 点击右侧的图钉图标，将其固定到工具栏

## 功能测试清单

### 核心功能测试

#### 1. 域名分组测试
**测试步骤**：
1. 打开多个不同域名的标签页（如 GitHub、YouTube、Stack Overflow）
2. 点击 Tab Manager 图标打开管理面板
3. 检查标签页是否按域名分组显示

**预期结果**：
- [x] 标签页按域名自动分组
- [x] 每个域名显示独立的卡片
- [x] 卡片显示标签页数量

#### 2. 域名置顶测试
**测试步骤**：
1. 在管理面板中，将鼠标悬浮在域名卡片上
2. 点击"置顶"按钮
3. 关闭管理面板
4. 重新打开管理面板

**预期结果**：
- [x] 域名卡片显示在"置顶域名"区域
- [x] 置顶状态持久化（关闭后重新打开仍然置顶）
- [x] 可以取消置顶

#### 3. 一键清理测试
**测试步骤**：
1. 在管理面板中，找到要关闭的域名卡片
2. 点击"关闭全部 X 个"按钮
3. 检查标签页是否被关闭

**预期结果**：
- [x] 域名分组的所有标签页被关闭
- [x] 卡片从管理面板中消失
- [x] 显示 Toast 提示

#### 4. 重复检测测试
**测试步骤**：
1. 打开多个相同的页面（相同 URL）
2. 打开管理面板
3. 检查是否显示重复标记

**预期结果**：
- [x] 重复标签页显示"X 个重复"标记
- [x] 可以点击"关闭 X 个重复"按钮去重
- [x] 保留一个标签页，关闭其他重复的

#### 5. 点击跳转测试
**测试步骤**：
1. 打开管理面板
2. 点击某个标签页标题
3. 检查是否跳转到对应标签页

**预期结果**：
- [x] 点击后跳转到对应标签页
- [x] 如果标签页在其他窗口，自动切换窗口
- [x] 管理面板保持打开状态

#### 6. 主题切换测试
**测试步骤**：
1. 打开管理面板
2. 点击顶部栏的主题切换按钮
3. 检查主题是否切换

**预期结果**：
- [x] 点击按钮切换主题（深色/浅色/跟随系统）
- [x] 主题设置持久化
- [x] 界面颜色相应变化

#### 7. 窗口合并测试
**测试步骤**：
1. 打开多个浏览器窗口
2. 打开管理面板
3. 点击"合并 X 个窗口"按钮

**预期结果**：
- [x] 所有窗口的标签页合并到当前窗口
- [x] 显示 Toast 提示
- [x] 管理面板刷新显示

#### 8. 休眠建议测试
**测试步骤**：
1. 打开一些标签页
2. 等待超过休眠阈值时间（默认 2 小时）
3. 打开管理面板

**预期结果**：
- [x] 显示休眠建议区域
- [x] 列出超过阈值未访问的标签页
- [x] 显示未访问时间
- [x] 可以关闭休眠标签页

#### 9. 搜索过滤测试
**测试步骤**：
1. 打开管理面板
2. 在搜索框中输入关键词
3. 检查标签页是否过滤

**预期结果**：
- [x] 实时过滤标签页
- [x] 支持搜索标题和 URL
- [x] 按 / 快捷键聚焦搜索框
- [x] 按 Escape 清除搜索

#### 10. 复制单个标签页链接测试
**测试步骤**：
1. 打开管理面板
2. 鼠标悬浮在某个标签页上
3. 点击复制按钮
4. 粘贴到文本编辑器

**预期结果**：
- [x] 鼠标悬浮时显示复制按钮
- [x] 点击复制按钮复制该标签页链接
- [x] 复制后显示成功提示

#### 11. 设置页面测试
**测试步骤**：
1. 打开管理面板
2. 点击设置按钮
3. 修改设置并保存

**预期结果**：
- [x] 显示设置页面
- [x] 可以修改主题、列数、休眠阈值等
- [x] 设置持久化保存
- [x] 可以重置设置

## 性能测试

### 1. 打开时间测试
**测试方法**：
1. 点击 Tab Manager 图标
2. 测量从点击到面板完全显示的时间

**预期结果**：
- [x] 打开时间 < 1 秒

### 2. 搜索响应测试
**测试方法**：
1. 打开管理面板
2. 输入搜索关键词
3. 测量从输入到结果显示的时间

**预期结果**：
- [x] 搜索响应时间 < 100ms

### 3. 动画流畅度测试
**测试方法**：
1. 打开管理面板
2. 关闭标签页或域名分组
3. 观察动画是否流畅

**预期结果**：
- [x] 动画流畅，无卡顿
- [x] 动画时间合理
